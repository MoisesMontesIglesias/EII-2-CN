Ejercicio 12¶

Construir, como array numpy, la segunda imagen mostrada debajo. La imagen tiene un tamaño de $2000 \times 2000$ pixels. En la primera imagen, el valor para cada pixel es la distancia euclídea al centro de la imagen $(c_x,c_y)$ al cuadrado.

$$ d = (c_x-j)^2+(c_y-i)^2 $$

La segunda imagen es la primera imagen invertida.

Mostrar el resultado en el terminal iPython. Guardar la imagen como esfera.jpg.

In [1]:
%matplotlib inline
from __future__ import division   
from PIL import Image             
import numpy as np                
import matplotlib.pyplot as plt

Generamos la primera imagen

In [2]:
t = 2000
cx = t/2. ; cy = t/2.;
M = np.zeros((t,t))

for i in range(t):
    for j in range(t):
        q = (i-cy)**2+(j-cy)**2
        if np.sqrt(q) < cx:
            M[i,j] = q

y la mostramos

In [3]:
plt.imshow(M,cmap='gray')
plt.show()

Invertimos la imagen y la mostramos

In [4]:
Ma = np.max(M)
M = Ma-M         
plt.imshow(M,cmap='gray')
plt.show()

Guardamos la imagen en un fichero

In [5]:
Ma = np.max(M)
mi = np.min(M)
M = (M-mi)/(Ma-mi)
M = M*255
M = M.astype(np.uint8)
I = Image.fromarray(M)
I.save('esfera.jpg')
